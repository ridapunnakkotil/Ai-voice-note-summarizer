# AI Voice Note Summarizer

A full-stack AI app that converts audio into structured insights.

## Features
- Upload audio
- Transcribe using Whisper (local)
- Generate summary, key points, action items

## Tech Stack
- React
- FastAPI
- Whisper (Local AI)

## How to run

Frontend:
cd client
npm install
npm start

Backend:
cd python-backend
venv\Scripts\activate
python -m uvicorn main:app --reload
